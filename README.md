# 幕後交易

# Intro your game
    #3-5人 回合制吹牛類型

    這是個大家一起參加可愛動物拍賣會的遊戲，每個人運用手上現有的金錢，透過拍賣喊價來購買動物，
    除了公開拍賣喊價之外，也可以和對方發起幕後交易，在交易的過程中，
    要小心對方騙人，同時也要專心騙對方的動物喔，
    遊戲最後必須以四張相同的動物來算分，獲得最高分的人獲勝。
